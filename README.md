# Seda-Generator

Archivage numérique - Digital archiving - ProBANT

Générateur automatique de bordereaux de transfert SEDA ProBANT
ProBANT signifie Production de bordereaux pour l'archivage numérique territorial

Ce projet est l'ensemble des exécutables du générateur
Disponible sur GitHub https://github.com/PatPercot/SedaGenerator-Distribution-Executables

Pour démarrer rapidement : lire le fichier HOWTO.txt
https://github.com/PatPercot/SedaGenerator-Distribution-Executables/blob/master/HOWTO.txt



Ce projet ne contient pas les sources du générateur.
Les projets contenant les fichiers sources du générateur sont dans les projets suivants :
	https://github.com/PatPercot/Seda-Generator
	https://github.com/PatPercot/Java-Seda-Generator
	https://github.com/CGI-France/Java-Seda-Generator
